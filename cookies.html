Cookie 政策

最後更新：2026-05-20｜營運方：真人股份有限公司

一、簡述

本儀表板（以下稱「本服務」）不使用追蹤型 Cookie。 我們僅在登入功能下使用瀏覽器內建的 localStorage 儲存登入會話（session token）、 不蒐集、不追蹤您在本服務以外的瀏覽行為。

二、什麼是 Cookie / localStorage

Cookie 是網站存放在您瀏覽器的小型文字檔、 通常用於記住登入狀態、語言偏好或追蹤使用行為。

localStorage 是現代瀏覽器提供的本地儲存機制、 資料只留在您的裝置內、不會自動隨每次連線傳送到伺服器（這跟 Cookie 不同）。

三、本服務使用哪些

3.1 必要技術項目（不可關閉、否則無法使用對應功能）

名稱類型用途保存期限
lxy-auth localStorage 儲存 Supabase 登入會話 token、讓您重新整理頁面後仍維持登入 登出時清除 / 一週自動失效

※ 僅在您於本服務點選「登入 / 註冊」並登入後產生。 未登入訪客的瀏覽器內、本項目不存在。

3.2 我們不使用的項目

  • Google Analytics / GA4 — 完全沒有
  • Meta Pixel（Facebook 廣告追蹤） — 完全沒有
  • 第三方廣告 cookie — 完全沒有（本服務不投放廣告）
  • 跨網站追蹤 cookie — 完全沒有
  • 使用者行為熱圖（Hotjar / Microsoft Clarity） — 完全沒有
  • Affiliate / 聯盟行銷 cookie — 完全沒有

3.3 第三方服務 cookie

本服務透過 CDN 載入以下第三方資源、彼等可能設定自家 cookie、不在本公司控管範圍：

  • jsdelivr.net（CDN）— 載入 Chart.js / Supabase JS、純技術 cookie
  • unpkg.com（CDN）— 載入 Leaflet 地圖元件
  • OpenStreetMap tile server — 載入地圖磚、無 cookie
  • Google News（新聞連結點擊後） — 點選新聞時導向 Google News 的 redirect URL、之後依 Google 自家 cookie 政策

若您對 CDN 提供的內容有疑慮、可在瀏覽器封鎖第三方 cookie、本服務功能不受影響。

四、如何管理 / 清除

4.1 清除本服務的 localStorage（等同登出）

  1. 登入後、點 header 右上「登出」按鈕、立即清除
  2. 或瀏覽器設定 → 隱私 → 清除瀏覽資料 → 勾選「網站資料」→ 套用至本服務網域

4.2 一般瀏覽器 Cookie 管理

  • Chrome：設定 → 隱私和安全性 → Cookie 和其他網站資料
  • Safari：偏好設定 → 隱私 → 管理網站資料
  • Firefox：選項 → 隱私與安全 → Cookie 及網站資料
  • Edge：設定 → Cookies 和網站權限

4.3 跨站追蹤封鎖

建議啟用瀏覽器「防止跨網站追蹤」設定（Safari / Firefox / Brave 內建）、 對保護您隱私有實質幫助、本服務功能不受影響。

五、Do Not Track（DNT）

本服務尊重您瀏覽器的「Do Not Track」設定。 但因本服務本身不追蹤、無論您是否啟用 DNT、效果相同。

六、變更通知

若本服務未來加入追蹤性或行銷性 cookie、會於本頁面公告變更內容、 並透過 dashboard 通知所有已登入用戶。 在您明示同意前、不會啟用任何追蹤性 cookie。
