一、權利歸屬概述

本儀表板（以下稱「本服務」）由真人股份有限公司開發、維運與發行。 以下說明各類內容之著作權歸屬與使用限制。

二、本公司擁有之著作權

以下內容之著作權皆歸真人股份有限公司所有：

  • 軟體程式碼：前端 JavaScript、後端 cron 腳本、Supabase RPC、資料庫 schema
  • 介面設計：版面配置、配色、icon、互動方式
  • 聚合分析結果：好感度分數演算法、議題敘事追蹤、地圖熱點計算、燈號規則
  • 視覺呈現：圖表配色、儀表板資訊架構
  • LLM 提示（prompts）與訓練資料：分類器之 system prompt、few-shot 範例選擇邏輯

未經本公司事先書面同意、不得複製、改作、散布、公開展示、商業使用上述內容。

三、第三方著作權

3.1 新聞內容

儀表板顯示之新聞標題、連結、發布時間等資訊、其著作權屬於原始新聞媒體所有。 點擊新聞連結會直接導向原始媒體網站、我們不重製新聞全文。

若您是新聞媒體代表、認為本儀表板對貴社新聞之引用方式有疑慮、 請聯絡 humaninterface2026@gmail.com、我們會在 7 個工作天內回覆並處理。

3.2 社群留言

本儀表板顯示之 Facebook / Instagram / Threads 公開留言、其著作權屬於原始留言者所有。 我們僅基於合理使用原則進行非商業性質之輿情分析。

若您是留言原作者、希望您的留言不出現於本儀表板、請聯絡我們、我們會將該則留言從顯示中移除。

3.3 LLM 模型

本儀表板使用之大型語言模型（含 OpenAI 等）之權利屬其原開發商所有、 我們僅依其 API 條款使用、未持有模型本身之著作權。

3.4 開源套件

本儀表板使用多項開源套件（Supabase JS / Chart.js / Leaflet 等）、依各自授權條款使用。

四、衍生作品說明

本儀表板之以下內容屬衍生作品、其著作權歸屬本公司、但仍尊重原始來源：

  • LLM 燈號標註（紅燈／黃燈／綠燈）— 對公開新聞標題的分類判讀
  • 人工修正紀錄— 由 admin 標記的判讀更正
  • 議題敘事追蹤— 議題熱度時序聚合
  • 地圖熱點— 城市事件聚類結果

這些衍生分析結果本身、未經書面授權不得商業性質之大量擷取或轉售。

五、合理使用

以下為本服務歡迎的使用方式（不需要事先申請）：

  • 學術研究：論文、報告中引用本儀表板內容、請註明來源「政治戰情儀表板」與本網址
  • 新聞報導：採訪或評論本儀表板分析結果、請註明資料來源
  • 個人觀察：在社群媒體截圖分享、請保留底部來源浮水印（若有）
  • 政策諮詢：智庫、政府機構內部討論時引用
